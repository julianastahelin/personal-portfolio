<h1>Personal portfolio</h1>

<p>This is my personal portfolio as a frontend web developer, where I show my CV and some recent projects. This project was built with React and Styled Components.</p>

<h2>Overview</h2>

<h3>The project</h3>
<p>My portfolio has a presentation section followed by CV, Projects and Contact sections, all of them in two languages (portuguese and english) that can be toggled in the menu. It works similarly to a landing page. It is responsive to screen sizes from mobile to desktop.</p>
